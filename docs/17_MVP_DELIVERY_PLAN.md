# 17. MVP Delivery Plan

This document defines the recommended MVP delivery sequence for the Wedding Surface Agent.

## MVP Goal

Deliver a thin but complete vertical slice: project intake, upload, agent job, minimal content generation, validation, and ZIP export.

## Milestones

1. Repository bootstrap
2. Supabase schema and RLS
3. Project CRUD and upload flow
4. Worker skeleton and minimum LangGraph path
5. Required 8 JSON export
6. 6-Gate QA
7. ZIP packaging

## Excluded from MVP

- Full growth dashboard
- Advanced AI visibility automation
- Billing
- Multi-industry support
- Full A/B testing
